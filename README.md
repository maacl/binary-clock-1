# binary-clock

A binary clock in Hoplon

## Dependencies

- java 1.7+
- [boot][1]
- [leiningen][2]

## Usage

You'll probably want to have two terminals open in your project's
directory.

1. Start the auto-compiler.

```bash
$ boot watch hoplon
```

2. Open the compiled html file.

```bash
$ cd resources/public
$ open index.html
```

3. You're already done.
